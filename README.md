## 🛠️ Installation  
```bash
# Cloner le dépôt
git clone https://github.com/Brnbt/portfolio.git

# Se déplacer dans le dossier
cd portfolio
```

Puis ouvrir le fichier `index.html` dans votre navigateur — aucune installation supplémentaire requise (site statique).
